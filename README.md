# basicCalculator
Csharp'ta metotları kullanarak basit bir hesap makinesi uygulaması yaptım. Hesap makinesiyle toplama, çıkarma, çarpma, bölme işlemleri yapılıyor. Ayrıca, Console.Clear() metoduyla kullanıcı işlem yaptıktan sonra ekran temizleniyor. Aynı zamanda işlem bittikten sonra kullanıcıya tekrar işlem yapmak isteyip istemediği soruluyor ve cevabına göre menüye tekrar dönülüyor ya da uygulamadan çıkış yapılıyor.

I made a simple calculator app using methods in CSharp. Addition, subtraction, multiplication and division operations are done with the calculator. Also, the screen is cleared after the users take action with the Console.Clear() method. At the same time, after the process is finished, users asked whether they want to perform the action again, and according to their answer, they return to the menu or exit the application.
